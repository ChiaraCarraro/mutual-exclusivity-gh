Das Spiel

Im folgenden Spiel sieht man immer ein Mädchen und 3 verschiedene Bilder.
Eine Stimme fördert das Kind auf, ein Bild auszuwählen.

Durch Klicken auf das Lautsprechersymbol können Sie die Stimme erneut abspielen. Sobald das Kind ein Bild ausgewählt hat, klicken Sie auf "WEITER", um fortzufahren.

Bitte klicken Sie jetzt auf "WEITER", um fortzufahren - oder auf das Lautsprechersymbol, um die Einführung für das Kind anzuhören.
